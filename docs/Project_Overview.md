# Project Overview

## Introduction 

The Change Management System is a web-based application developed to simplify and manage change requests in an organization. It provides a centralized platform where users can submit requests, track their status, and communicate with developers efficiently.

## Purpose 

The main purpose of this project is to improve the change management process by providing transparency, faster approvals, and better coordination between users and developers.

## Scope

This system includes user authentication, dashboard, ticket management, developer assignment, approval workflow, and status tracking. It is dedsigned for educational and internship purposes.
