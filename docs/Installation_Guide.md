# Installation Guide

## Prerequisites

- Python 3.x
- Node.js
- npm
- Git
- Visual Studio Code

## Installation Steps

1. Clone the repository.
2. Install backend dependencies.
3. Install frontend dependencies.
4. Start the Django server.
5. Start the React development server.

## Backend

```bash
cd backend
python manage.py runserver
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
