# User Manual

## Introduction

The Change Management System is designed to help users submit, manage, and track change requests efficiently.

## User Guide

### Step 1: Login
Users log in using their registered credentials.

### Step 2: Dashboard
After login, users can view the dashboard containing all available options.

### Step 3: Raise Change Request
Users can submit a new change request by filling in the required details.

### Step 4: Track Requests
Users can monitor the status of submitted requests.

### Step 5: Approval Process
Administrators review and approve or reject the submitted requests.

### Step 6: Developer Assignment
Approved requests are assigned to developers for implementation.
